Startpage

This app is based on the minimalistic CRUD App that shows how operations on CouchDB are made with the Kanso Framework.
You can find it on github

So whats new here?

  • Use of widgets in lib/types.js
  • Included CSS in static/style.css
  • Rearranged the templates skeleton in templates/*.html
    Note:
    Do not use "}", "{" or ":" within HTML comments in templates. This will cause problems, which are difficult to track down.
    Use Dust's template comments instead: "Comments, which do not appear in template output, begin and end with a bang (!)"
  • Integrated jquery UI (jquery-ui-1.8.16.custom.min.js) and replaced jquery jquery-1.5.2.min.js through jquery-1.6.2.min.js
    (This is what you get, if you go to http://www.jqueryui.com and do a download.)
  • Demonstration of event handler in the Footer (for content that doesn't change)
  • Demonstration of event handler jQuery UI Demo (for content that does change)
